Anthropic has rolled out Shared Context for its Claude Excel and PowerPoint plugins, allowing the AI to maintain conversation history and share data between both applications simultaneously. This moves Claude beyond isolated single-app assistance toward a unified workspace where financial analysis can flow seamlessly from spreadsheet creation to presentation building.

---

- **Cross-app functionality** Claude can now access custom skills and full conversation context across both Excel and PowerPoint, ending the previous limitation of isolated sessions per application

- **Automated workflow integration** Pre-built skills like DCF modeling and data cleaning in Excel, plus custom style guides and writing voice skills, can be automatically triggered across both platforms

- **Real-world performance validation** Testing with Pershing Square Holdings data showed Claude successfully extracted 13 monthly return data points from PDFs, calculated Sharpe ratios, and created multi-slide presentations with charts and analysis

- **Data transfer capabilities** The system can move tables and charts between Excel and PowerPoint, though the process occasionally requires troubleshooting and manual prompting when requests get stuck

- **Design limitations persist** While functionally capable, PowerPoint outputs still appear "AI-generated" and may not meet client-facing presentation standards despite using custom style guides

- **Agentic evolution signal** This integration represents a clear shift from simple chat-based AI assistance toward autonomous agents capable of managing complex, multi-application workflows

- **Enterprise positioning** Anthropic is positioning Claude Cowork as a complete operating system for knowledge work, targeting financial analysis, business development, and data synthesis tasks
