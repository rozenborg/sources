## Summary

- Claude Opus 4.6 launches with enhanced coding capabilities, sustained agentic task performance, and improved code review/debugging skills

- First Opus-class model featuring 1M token context window in beta with 128k output token support

- Achieves state-of-the-art performance on Terminal-Bench 2.0 (agentic coding) and outperforms GPT-5.2 by 144 Elo points on GDPval-AA economic tasks

- Pricing remains unchanged at $5/$25 per million tokens; premium pricing ($10/$37.50) applies for prompts exceeding 200k tokens

- New adaptive thinking feature allows model to determine when deeper reasoning is needed, with four effort levels (low, medium, high, max)

- Context compaction (beta) enables automatic summarization of older context to support longer-running tasks without hitting limits

- Agent teams capability in Claude Code allows multiple agents to work in parallel and coordinate autonomously

- Enhanced Excel integration with improved long-running task performance and multi-step change handling in single pass

- PowerPoint integration (research preview) reads layouts, fonts, and slide masters to maintain brand consistency

- US-only inference option available at 1.1× token pricing for compliance-sensitive workloads

- Safety profile matches or exceeds other frontier models with low rates of misaligned behavior and reduced over-refusal rates

- Partners report successful autonomous handling of complex tasks including 50-person organization management across 6 repositories

- 76% score on 8-needle 1M variant MRCR v2 benchmark vs 18.5% for Sonnet 4.5, indicating superior long-context performance

- Six new cybersecurity probes implemented to detect harmful responses given model's enhanced cybersecurity capabilities

- Model demonstrates 38/40 superior performance rate in blind cybersecurity investigations vs Claude 4.5
