Anthropic releases Claude Opus 4.8, delivering the first AI model to achieve 100% completion on Super-Agent benchmarks while cutting fast mode pricing by three times and introducing dynamic workflows that can manage hundreds of parallel subagents for enterprise-scale coding tasks.

---

- **Benchmark leadership** Opus 4.8 is the only model to complete every case end-to-end on Super-Agent benchmark, beating previous Opus versions and GPT-5.5 at cost parity, with 84% score on Online-Mind2Web for computer use tasks

- **Enhanced reliability** Model shows 4x improvement in code quality self-assessment and significantly better "honesty" with reduced unsupported claims, while achieving new highs in prosocial alignment traits

- **Dynamic workflows capability** New feature allows Claude Code to run hundreds of parallel subagents in single sessions for codebase-scale migrations across hundreds of thousands of lines of code from start to deployment

- **Effort control system** Users can now adjust computational effort levels, with high effort as default providing similar token usage to Opus 4.7 but better performance, plus "extra" and "max" settings for complex tasks

- **Cost optimization** Fast mode pricing reduced to $10/$50 per million input/output tokens (3x cheaper than previous models) while maintaining $5/$25 standard pricing unchanged from Opus 4.7

- **Enterprise adoption momentum** Strong validation from major enterprise users including Databricks, Hebbia, and legal platforms reporting meaningful improvements in consistency, reasoning quality, and 61% lower token costs for multimodal tasks

- **Strategic roadmap preview** Anthropic signals upcoming release of higher-intelligence "Mythos-class" models from Project Glasswing currently in cybersecurity testing, pending enhanced safety safeguards development
