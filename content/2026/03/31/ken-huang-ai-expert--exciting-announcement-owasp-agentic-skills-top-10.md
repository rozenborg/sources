OWASP has officially launched the Agentic Skills Top 10 (AST10) project, the first comprehensive security framework addressing critical risks in AI agent skills across platforms, responding to active attacks on the AI agent ecosystem as of Q1 2026. This framework provides evidence-based mitigations for the 10 most critical security vulnerabilities affecting AI agents, with dedicated guidance for security teams, skill developers, and platform developers.

---

- **Active Threat Landscape** The AI agent skill ecosystem is under active attack as of Q1 2026, making this security framework immediately relevant for organizations deploying AI agents

- **Comprehensive Framework** AST10 documents the 10 most critical security risks in agentic AI skills with evidence-based mitigations, security assessment checklists, and a universal skill format specification

- **Multi-Stakeholder Approach** Framework provides specific guidance for three key audiences: security teams (assessment and monitoring), skill developers (secure coding and signing), and platform developers (registry security and runtime isolation)

- **Research-Backed Intelligence** Project incorporates threat intelligence from multiple sources including Snyk ToxicSkills audit, Check Point Research on Claude Code vulnerabilities, and Antiy CERT's ClawHavoc campaign analysis

- **Actionable Security Controls** Key recommendations include implementing least-privilege access, cryptographic skill signing, sandboxed execution environments, and explicit user confirmation for privileged operations

- **Community-Driven Development** Project accepts contributions through GitHub issues, pull requests, and an interactive web form for submitting new risk entries, with plans for translations and localization

- **Immediate Implementation** Security teams can begin using the assessment checklist and platform-specific guidance immediately, while developers can adopt the universal skill format and secure coding practices
