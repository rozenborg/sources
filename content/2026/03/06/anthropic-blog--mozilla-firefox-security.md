Anthropic's Claude AI discovered 22 vulnerabilities in Mozilla Firefox within two weeks, with 14 classified as high-severity—representing nearly a fifth of all high-severity Firefox vulnerabilities fixed in 2025. While Claude excels at finding and fixing security flaws, it struggles to exploit them, creating a critical window where defenders have the advantage before AI capabilities potentially balance out.

---

- **Vulnerability Discovery Scale** Claude Opus 4.6 scanned nearly 6,000 C++ files and submitted 112 unique reports to Mozilla, demonstrating AI's ability to accelerate security research at unprecedented speed compared to traditional human-led efforts.

- **Exploitation Gap** Despite spending $4,000 in API credits across hundreds of attempts, Claude only successfully created working exploits in two cases, indicating current AI limitations in weaponizing discovered vulnerabilities.

- **Enterprise Security Tool** Anthropic launched Claude Code Security in limited research preview, bringing automated vulnerability discovery and patching capabilities directly to customers and open-source maintainers.

- **Task Verifier Framework** Claude works best when paired with automated verification tools that can test whether vulnerabilities are actually fixed and program functionality is preserved, providing a blueprint for AI-assisted security workflows.

- **Defensive Window Closing** The company warns that the current advantage for defenders may be temporary, as rapid AI progress suggests future models will likely close the gap between vulnerability discovery and exploitation capabilities.

- **Industry Collaboration Model** Mozilla's partnership approach—accepting bulk submissions with minimal test cases, detailed proofs-of-concept, and candidate patches—provides a template for how organizations can effectively work with AI security researchers.

- **Broader Impact Scope** Beyond Firefox, Claude has identified vulnerabilities in other critical infrastructure including the Linux kernel, indicating the technology's potential for system-wide security improvements across the open-source ecosystem.
