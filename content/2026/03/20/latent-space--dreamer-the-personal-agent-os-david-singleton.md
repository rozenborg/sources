Dreamer, led by former Stripe CTO David Singleton and Android veteran Hugo Barra, has launched as a consumer-first platform where anyone can discover, build, and use AI agents without coding experience. The core innovation is "Sidekick," a personal agent that helps users customize and create other agents through natural language, backed by a full-stack development platform that includes hosting, databases, serverless functions, and revenue sharing for tool builders.

---

- **Consumer-first agent platform** - Unlike technical platforms, Dreamer targets non-technical users like Singleton's "non-technical but smart sister" who can install community-built agents and customize them through natural language conversations with Sidekick

- **Full-stack infrastructure** - Platform includes hosting, databases, prompt management, serverless functions, and VM deployment, eliminating typical setup friction that requires multiple service providers and API keys

- **Revenue-sharing ecosystem** - Tool builders get paid proportionally based on usage of their integrations, with premium tools like Parallel Web Systems operating on per-use billing while maintaining free trials for agent builders

- **Agent-building-agents architecture** - Sidekick creates a plan, builds the application, tests it automatically, and iterates in a feedback loop, with builds taking 10-15 minutes initially but producing working applications most of the time

- **Cross-platform integration** - Agents can output to existing apps users already use (like Apple Podcasts for daily briefings) rather than forcing users into new interfaces, with mobile and desktop experiences

- **Real production usage** - The company runs internally on Dreamer agents for operations like waitlist management and lead qualification, demonstrating dogfooding at scale

- **Enterprise penetration strategy** - Seeing adoption patterns where individuals build personal productivity agents then share with family members and work teams, creating organic workplace infiltration

- **High-quality data integration** - Platform includes direct API feeds (not scraped data) for sports, news, and other domains, enabling more reliable agent behaviors than web-scraping approaches
